1-click deployments and hosted platforms

Kimai can be hosted on various systems and many hosting companies support 1-click installations for Kimai directly.

Hosting Kimai this way is neither officially recommended nor supported. But in the best interest of the community, this quick-starter is provided to help you set up Kimai.

Self-hosting knowledge prerequisites

Self-hosting Kimai requires technical knowledge, including:

  • Setting up and configuring servers and containers
  • Managing application resources and scaling
  • Securing servers and applications
  • Configuring Kimai

Kimai recommends self-hosting for expert users. Mistakes can lead to data loss, security issues, and downtime. If you arenâ€™t experienced at managing servers, Kimai recommends the hosted cloud.

The following platforms adopted Kimai to be compatible with their one-click installation systems.

YunoHost

With YunoHost, you can easily manage a server for your friends, association or enterprise. The Kimai package can be found at GitHub.

Install Kimai with YunoHost

Cloudron

Cloudron provides a secure and ready to use Kimai package, which will be kept up-to-date automatically. There is documentation available on how to customize configs and install plugins.

Install Kimai with Cloudron

Cloudjiffy

CloudJiffy provides a scalable, hourly billed and easy to use PaaS platform. Kimai is always deployed from the latest Github branch, thus you can rest easy that your software will always be up-to-date.

Kimai is only a button-click away

Elestio

Elestio offers fully managed Kimai instances in 112 regions across 28 countries and covers everything from deployment to security, monitoring, and updates. You get full root SSH access to Kimai instance, and a web dashboard for stack and env var changes. You donâ€™t have to worry about updates as Elestio auto updates Kimai instance weekly which can be configured or disabled from the dashboard easily.

Deploy Kimai to Elestio

Softaculous

Be warned that Softaculous does not yet provide updates of Kimai, so you are stuck with the installed version. There have been a few other problems reported, using this installation type is ONLY FOR EVALUATION usage.

Install Kimai with Softaculous

Vesta Control Panel

Be aware that VestaCP uses the admin user instead of www-data. Replace the names in the regular permission commands. Read this issue if you have further questions.

ISPConfig 3

There is an installation doc (only in German) available at www.howtoforge.de.
